# cryptostack
cryptostack is a mobile application to track crypto trade activities and tax auditing.

Tech Stack:
	• Frontend: Next.js (App Router), Tailwind CSS, TanStack Table.
	• Backend: Node.js (Next.js API Routes), PostgreSQL (Supabase/Neon), Prisma ORM.
	• Math: decimal.js for all financial calculations to avoid floating-point errors.
  • Data: CoinGecko API for CAD price history.
